**「AI 好同事三部曲」第三堂（收官）。** 120 分鐘實作工作坊：看懂 AI 為什麼越講越遠、越講越篤定——診斷走鐘、救回對話、幫團隊定求證準則。

## 一句話說完這門課

> **AI 不是不聽話，是聽了太多話。**

它從整段對話接著寫，不是只回應你最後一句。看懂它正在延續什麼——導正、重開、求證、交回，四個動作把工作拉回來。

## 使用說明

- **課堂學員**：跟著講師走，每人先做 Level 1，再依程度選 Level 2 或 3。
- **自學者**：可獨立學習；用「練習對話文本」即可完成全部練習。
- **列印**：題卡與工具表已針對 A4 排版，瀏覽器直接列印，每份一頁。
- **深／淺色**：頁面自動配合裝置模式；列印一律輸出淺色。

## 三個誠實的原則

1. **工具不限**——ChatGPT、Gemini、Claude、Copilot 都行，**所有練習皆可在免費版完成**。
2. **誠實的比喻**——課程的「語域地形」是教學比喻，不是模型內部構造；比喻是我們的，現象是真的。
3. **流暢不等於為真**——認識它之後，它還是會錯；求證與交回，永遠是四動作的後兩個。

Includes three tiered exercise cards, a four-actions quick-reference sheet (steer / reset / verify / transfer), fictional practice dialogues, and the slide deck. All exercises work on free tiers.
